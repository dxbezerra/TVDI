As etapas para a obtenção do TVDI consistem em:
1.   Download das Imagens
2.   Extração das bandas de interesse das imagens
3.   Mosaicagem
4.   Recorte para área de interesse
5.   Acesso das imagens
6.   Tratamento dos dados
7.   Cálculo das retas
8.   Cálculo do TVDI
